angular-loading-counter

An AngularJS module to count time consuming tasks and show a visual feedback while those are performed. To quick start it, inject the dsg.loadingCounter module in you angular application and push the LoadingCounter service through your http interceptors:

  var app = angular.module('yourAngularApp', ['dsg.loadingCounter']);
  
  app.config(function ($httpProvider) {
    $httpProvider.interceptors.push('LoadingCounter');
  });

You can configure the LoadingCounter service to exclude some URLs from being counted:

  angular
    app.config(function ($httpProvider, LoadingCounterProvider) {
    
      // Configure an exclusion for HTTP GET paths starting with `/api/utils/country`
      LoadingCounterProvider.addExclusion(/^\/api\/utils\/country/i, /get/i);
     
      // Push LoadingCounter in the http interceptors
      $httpProvider.interceptors.push('LoadingCounter');
    });

Then add the loadingCounter directive in your index.html page:

  <html>
    ...
    <body ng-app="yourAngularApp">
      ...
      <div class="spinner" loading-counter>
        Loading...
      </div>
      ...
    </body>
  </html>

loadingCounter directive advanced usage:

  <div class="spinner" 
       loading-counter 
       hidden-class="hidden" 
       pending-tasks-count="tasks">
    <div>Loading...</div>
    <div>Still have to finish {{$parent.tasks}} tasks</div>
  </div>
