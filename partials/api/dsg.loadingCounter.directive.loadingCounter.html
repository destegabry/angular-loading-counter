loadingCounter
directive in module dsg.loadingCounter

Description

Directive which hides a DOM element until the LoadingCounter service has a pendingRequestsCount grater than 0.

The hiding is performed changing element's CSS property display until you specify hiddenClass and/or shownClass attributes to the element.

You could also use the pendingTasksCount binding to show a countdown or some sort of progress bar.

Dependencies

  • LoadingCounter

Usage

as element:
<loading-counter
       [hidden-class="{string}"]
       [shown-class="{string}"]
       [pending-tasks-count="{number}"]>
</loading-counter>
as attribute
<ANY loading-counter
     [hidden-class="{string}"]
     [shown-class="{string}"]
     [pending-tasks-count="{number}"]>
   ...
</ANY>
as class
<ANY class="loading-counter [hidden-class: {string};] [shown-class: {string};] [pending-tasks-count: {number};]">
   ...
</ANY>

Directive info

  • This directive creates new scope.
